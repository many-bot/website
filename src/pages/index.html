O que é o ManyBot?
O ManyBot é um ecossistema de automação de mensageria para negócios e comunidades online. Sua arquitetura de microkernel permite com que cada funcionalidade seja um plugin independente - você controla o que o seu bot faz.

O projeto é de código aberto e é mantido pela comunidade. Contribuições são bem-vindas.
