# Novo plugin: ManyMedia

Hoje, um novo plugin foi adicionado ao repositório do ManyPlug, o "ManyMedia". 

Esse plugin visa unificar os plugins "video" e "audio" em apenas um só, além de resolver diversos bugs que os antigos tinham ao baixar vídeos do Reddit e Instagram.

> Importante: caso esteja com algum desses plugins instalados e deseja utilizar o ManyMedia, é recomendado removê-los antes disso para evitar conflitos.

O plugin estará recebendo mais atualizações em breve para garantir compatibilidade absoluta com qualquer link de qualquer rede social da Internet.

Os comandos para utilizá-lo continua sendo o mesmo de "video" e "audio". Exemplo:

```
!video https://example.com/...
```

e:

```
!audio https://example.com/...
```

---

Para fazer a instalação do plugin no seu projeto ManyBot, use o seguinte comando:

```
manyplug install manymedia
```

E ative:

```
manyplug enable manymedia
```

> Verifique dependencias antes de usar.
